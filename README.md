## Instalation

```
npm install
```

## Start local node

```
anvil
```

## Deploy contracts to local node

In another terminal execute:

```
npm run deploy-contracts
```

\*Make sure the OWNER_PK env variable has the 'export' word in the line start

## Generate typechain types for TypeScript usage

```
npm run postinstall
```

## Run script

```
npm start
```
